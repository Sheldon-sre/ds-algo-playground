# 递归与分治思维

* 目标：掌握递归建模能力，这是算法的核心思维跃升

递归的本质是：
    把一个大问题，拆解成结构相同的小问题

    听起来简单，但真正掌握它之后，树、图、动态规划全都会豁然开朗。
